# Phase 4: 视觉资产与图像生成

## 提示词模板

### 主场景设定

> 我需要为漫画设定一个主要场景的视觉资产库。
> 场景描述：[描述场景，如：杂乱的三人办公室]
> 请提供：
> - 固定视角和布局描述
> - 详细的环境细节（杂物程度、光照状态）
> - Nano Banana 2英文生图Prompt（适用于黑白漫画风格）

### 核心道具定义

> 我需要为漫画设定一个具有象征意义的专属道具。
> 道具描述：[描述道具，如：显示"ERROR"警告的魔法平板]
> 请提供：
> - 道具的详细外观描述
> - 道具的象征意义
> - Nano Banana 2英文生图Prompt

### 氛围网点SOP

> 我需要为特定情绪设计氛围背景。
> 情绪类型：[如：绝望/搞笑/热血]
> 请提供：
> - 情绪的视觉化表现方式
> - 网点/线条的运用建议
> - Nano Banana 2英文生图Prompt（无角色，只表现情绪氛围）

## Nano Banana 2 万能公式

```
[画风基调] + [角色视觉锚点] + [动作表情] + [场景特效]
```

**画风基调（固定不变）**：
```
Vertical black and white Japanese manga page, screentone shading, high contrast line art, single manga panel, monochrome,
```

## 输出要求

- 场景要固定视角和细节
- 道具要有象征意义
- 氛围要明确、有力
- 所有 Prompt 必须适合黑白漫画风格
