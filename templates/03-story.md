# 剧情引擎与脚本模版 (Story Engine)

---

## 模块A：生成单卷大纲

### 复制发给AI：

```
请基于我的《02_IP_Bible》设定，为第一卷（第1-10话）设计主线大纲。
要求：
1. 明确本卷主角的终极小目标
2. 列出1-10话的单话核心事件，每话必须有一个小阻碍
3. 第10话结尾必须设置一个颠覆性的巨大悬念（Cliffhanger）
```

---

## 模块B：单话分镜脚本转化

### 复制发给AI：

```
请将第[X]话的大纲转化为可执行的漫画分镜脚本（设定为6页长条滑动格式）。
要求按照以下格式严格输出：

【P1】
- 构图与画面：（特写/中景/全景）描述画面内容、角色动作、背景
- 对白/内心独白：（极简、网感）
- 效果音SFX：（如：砰、叹气）

（注：最后一页必须制造情绪高潮或搞笑反转）
```
